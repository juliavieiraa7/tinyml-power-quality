## Python training pipeline

This folder contains the scripts and notebooks used for data generation, feature extraction, training and evaluation of the machine learning models.

### Contents

- Synthetic signal generation and feature extraction
- Training and evaluation of classifiers
- Validation using the SEED-PQD dataset
- Export of trained models for embedded deployment

The notebooks are intended for reproducibility of the results reported in the article.

## Pipeline de treinamento em Python

Esta pasta contém os scripts e notebooks utilizados para a geração de dados, extração de características, treinamento e avaliação dos modelos de aprendizado de máquina.

### Conteúdo

- Geração de sinais sintéticos e extração de características
- Treinamento e avaliação de classificadores
- Validação utilizando a base de dados SEED-PQD
- Exportação dos modelos treinados para implantação embarcada

Os notebooks permitem a reprodutibilidade dos resultados apresentados no artigo.
