## Embedded implementation (ESP32)

This folder contains the embedded implementation of the TinyML classifiers for power quality disturbance detection using an ESP32 microcontroller.

### Contents

- TensorFlow Lite model embedded as a header file
- Inference code using the EloquentTinyML library
- Functional validation of the inference pipeline

The embedded tests focus on validating correct model execution under memory and processing constraints.

## Implementação embarcada (ESP32)

Esta pasta contém a implementação embarcada dos classificadores TinyML para detecção de distúrbios de Qualidade de Energia utilizando um microcontrolador ESP32.

### Conteúdo

- Modelo TensorFlow Lite incorporado como arquivo de cabeçalho
- Código de inferência utilizando a biblioteca EloquentTinyML
- Validação funcional do pipeline de inferência

Os testes embarcados concentram-se na verificação da execução correta do modelo sob restrições de memória e processamento.
